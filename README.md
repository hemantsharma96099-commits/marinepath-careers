# MarinePath Careers

This is a Next.js + Tailwind starter for the MarinePath Careers landing page.

## How to run locally

1. Install dependencies
```bash
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
npm start
```
